# 📊 Sales Data Analysis

##  Overview
This project analyzes sales data to identify trends, patterns, and business insights.

##  Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI

##  Key Tasks
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- SQL queries for insights
- Dashboard creation

##  Key Insights
- Identified top-performing products
- Analyzed monthly revenue trends
- Found customer purchase patterns

##  Outcome
Created an interactive dashboard and generated insights to support business decisions.
